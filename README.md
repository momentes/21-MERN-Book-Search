# Module-21-Challenge
Bootcamp Module 21 Challenge - Model-View-Controller (MVC) Challenge: Tech Blog



# User Story
- AS A developer who writes about tech


# Acceptance Criteria
- GIVEN a CMS-style blog site
- WHEN I visit the site for the first time


# Installation

run

```
npm install
```

on the root folder.

to run the application

```
npm run develop
```
